# YandexMusic
 Описание проекта
Данный проект представляет собой адаптивный клон популярного музыкального сервиса Яндекс Музыка. Приложение создано с нуля/на основе базовой концепции, но переработано в полноценный самостоятельный сервис с уникальными доработками, улучшенным интерфейсом и расширенным функционалом для авторизованных пользователей.
Основная цель проекта — воссоздать привычный и удобный пользовательский опыт прослушивания музыки, добавив ключевые фичи для управления аккаунтом и персонализации контента
Ключевые фичи и реализованный функционал
В рамках разработки мы полностью реализовали и внедрили следующий функционал:
Система аутентификации (Полный цикл):
Регистрация новых пользователей.
Вход в существующий аккаунт.
Безопасный выход из системы (Sign Out) с очисткой сессии.
 Продвинутая навигация:
Удобное перемещение между основными экранами и разделами приложения без потери состояния плеера.
 Уникальный дизайн:
Современный, эстетичный и интуитивно понятный интерфейс, вдохновленный оригиналом, но адаптированный под наши решения.
 Полноценный аудиоплеер:
Стабильное прослушивание треков в реальном времени.
Возможность переключения музыки (вперед/назад), пауза и воспроизведение.
 Интеграция «Моей Волны»:
Умная система уведомлений, которая сообщает пользователю о том, что в данный момент играет его персональный поток («Моя Волна»).
 Технологический стек
База данных: Firebase
